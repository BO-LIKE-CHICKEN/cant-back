## 08.설명하는 변수

> 표현식의 의도가 드러나도록 변수 이름을 만들어 할당해보세요

1. 한글 변수 사용하기

   - 영어로 나타내기에 복잡한 변수인 경우에는 한글 변수를 사용한다면 쉽게 변수를 파악할 수 있다.

   - [세종대왕 프로젝트 (한글 코딩 컨벤션)](https://tosspayments-dev.oopy.io/chapters/frontend/posts/hangul-coding-convention)

   ```jsx
   const 국가지방자치단체_공공단체_금융사_여부 = false;
   ```

2. prefix, postfix로 변수 나타내기

   ```jsx
   // get-
   const getItems = () => {
    ...
   }

   // is-
   const isNumber = true
   ```
