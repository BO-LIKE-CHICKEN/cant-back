# CHAPTER 03. 대칭으로 맞추기

- 책의 초기화 지연 예시의 문제는 자바스크립트 const 키워드로 해결할 수 있다.

```js
const x = 3;

x = 4; // error
```

<br />

> 한 가지 패턴의 코드는 읽다 보면 그 방식에 금방 익숙해집니다 ... 한 가지 방식을 선택해서 정합니다. 다른 방식으로 작성한 코드를 선택한 방식으로 고칩니다.

- 팀 내부적으로 코딩 컨벤션을 정한다

```js

// 천 단위로 _를 사용한다
const value = 1000000 // Bad
const value = 1_000_000 // Good


// if-else 대신 early return을 사용한다

// Good
function example {
	if (조건) {
		return 코드1;
	}

	return 코드2;
}

// Bad
function example {
	if (조건) {
		return 코드1;
	} else {
		return 코드2;
	}
}
```
